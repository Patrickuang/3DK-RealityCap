translationFromPoint:toPoint:

Compute the translation between two points.

+ (RCTranslation *)translationFromPoint:(RCPoint *)fromPoint toPoint:(RCPoint *)toPoint

Parameters

fromPoint

The first RCPoint.

toPoint

The second RCPoint.

Return Value

A new RCTranslation object that such that calling [RCTranslation transformPoint:fromPoint] would return a point with the same 3D position as toPoint.
