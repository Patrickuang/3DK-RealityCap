Overview

This is a video preview view that can be used as an alternative to a AVCaptureVideoPreviewLayer. You can pass individual video frames to this class, and it will display them as they are received. One advantage of this design is that you can pass in video frames that you receive via [RCSensorFusionDelegate sensorFusionDidUpdateData:], which will ensure that the video and any augmented reality overlays you draw on top of it stay in sync.
